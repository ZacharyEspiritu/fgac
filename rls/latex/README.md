# `latex`

For convenience, we provide a minimal LaTeX wrapper around the figures produced
by the artifact so you can see how the generated LaTeX code renders without
having to set up your own LaTeX document.

## Usage

Compile the LaTeX artifact manually with:

```bash
bash latex/compile.sh
```

If you are using the top-level run commands, you should not have to call this
directly as the top-level commands call this after they produce figures.
